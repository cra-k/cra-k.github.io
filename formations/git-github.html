    CRA, ACEM Kénitra

    Présentation

    Git et GitHub ne sont pas la même chose. En effet, Git est système de contrle de version open source créé en 2005 par des developeurs travaillant sur le système d'exploitation Linux. Quand a GitHub, c'est une entreprise créée en 2008 qui en gros crée des outils qui s'intègre bien avec Git.

    Vous pouvez utiliser Git sans GitHub, cependant vous ne pouvez pas utiliser Git sans GitHub. Il faut savoir qu'il existe d'autres alternatives à GitHub à savoir GitLab, BitBucket, ou bien vous pouvez même créer votre propre serveur git. Ces derniers sont reconnues par Git sous le nom de Remote (serveur distant). Vous n'avez pas besoins d'un remote pour utiliser git, mais ça aide au partage de votre code avec les autres avec une très grande facilité.
